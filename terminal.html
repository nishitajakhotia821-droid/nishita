Integrated Terminal in VS Code

The integrated terminal in Visual Studio Code allows you to run commands, manage projects, install packages, and execute programs directly inside the editor without switching windows.

Start Learning Common Commands
Overview

What is the Terminal?

The terminal is a command-line interface built into VS Code. It allows direct interaction with your operating system and development tools.

Built-in Access

No need to open external terminal apps.

Faster Workflow

Run commands instantly inside your editor.

Multiple Terminals

Open and manage several terminals at once.

Commands

Common Terminal Commands

Navigate Folders

cd folder-name → move into folder
cd .. → go back

List Files

dir (Windows) or ls (Mac/Linux)

Install Packages

npm install package-name
pip install package-name

Run Programs

node app.js
python app.py

Shell Types

Supported Terminals

Command Prompt

Default Windows shell for basic commands.

PowerShell

Advanced Windows automation shell.

Bash

Common shell for Linux and Mac systems.

Cloud Shell

Used for cloud-based development environments.
