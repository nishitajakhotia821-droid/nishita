Complete Installation Guide

Install Visual Studio Code on Any Operating System

This guide explains every step required to install Visual Studio Code on Windows, macOS, and Linux. It also covers system requirements, portable versions, PATH configuration, updating, offline installation, common installation errors, and troubleshooting.

Download Guide Windows Setup

Download Visual Studio Code

Always download Visual Studio Code from Microsoft's official website to ensure you receive the latest stable version with security updates.

Windows

Supports Windows 10 and Windows 11 in both 64-bit and ARM64 editions.

macOS

Compatible with Intel and Apple Silicon Macs running modern macOS versions.

Linux

Available as .deb, .rpm, Snap package, and universal tar archive.

Install VS Code on Windows

Installing Visual Studio Code on Windows is straightforward and usually takes less than five minutes.

01

Download Installer

Download the latest Windows installer (.exe) from the official website.

02

Run Setup

Double-click the installer and allow Windows User Account Control if prompted.

03

Accept License

Read and accept the Microsoft License Agreement.

04

Select Installation Folder

Choose the default folder or specify another installation location.

05

Create Shortcuts

Enable Desktop Icon and Start Menu shortcuts for quick access.

06

Add to PATH

Select "Add to PATH" so VS Code can be launched using the code command.

07

Install

Click the Install button and wait while files are copied.

08

Launch VS Code

Open Visual Studio Code and begin creating your first project.

Windows Installation Tips

Enable PATH

This allows you to launch VS Code directly from Command Prompt or PowerShell by typing code.

Open with Right Click

Enable Explorer context menu integration for quick access.

Administrator Rights

Install as Administrator only if required by your organization.

Latest Version

Always install the newest stable release to receive security updates and new features.

Install VS Code on macOS

Visual Studio Code supports both Intel and Apple Silicon (M1, M2, M3, M4) Macs. Installation is simple and does not require advanced technical knowledge.

01

Download

Download the macOS ZIP package from the official Visual Studio Code website.

02

Extract ZIP

Open the downloaded ZIP file to extract the Visual Studio Code application.

03

Move to Applications

Drag Visual Studio Code into the Applications folder.

04

Launch

Open VS Code from Launchpad or the Applications folder.

05

Install Shell Command

Open the Command Palette and install the code command into your PATH.

06

Ready to Use

Begin installing extensions and creating projects.

Verify Installation

After installation, verify that Visual Studio Code has been installed correctly.

Open VS Code

Launch Visual Studio Code from the desktop shortcut, Start Menu, or Applications folder.

Test Terminal

Open Terminal and run the code --version command.

Open Folder

Create a folder and open it using the File → Open Folder option.

Install Extension

Install Live Server or Prettier to confirm Marketplace connectivity.

Common Installation Errors

Error Reason Solution
code is not recognized PATH variable missing Add VS Code to PATH and restart terminal.
Installer won't open Corrupted download Download installer again.
Permission Denied Administrator restriction Run installer as Administrator.
Extension Download Failed No Internet Check firewall or internet connection.
Slow Startup Too many extensions Disable unused extensions.
Blank Window Graphics issue Update graphics driver.

Troubleshooting Tips

Internet Connection

Ensure your computer has a stable internet connection before downloading updates and extensions.

Firewall

Corporate firewalls may block Marketplace downloads. Contact your network administrator if required.

Restart Computer

Restart the system after installation if PATH changes are not detected immediately.

Reinstall

If installation files become corrupted, uninstall VS Code and install the latest version again.

Free Disk Space

Keep enough storage available for updates, extensions and project files.

Use Stable Release

Beginners should install the Stable version instead of Insider builds.
